# ChatBot
This is a small React-based chatbot project built just for fun and practice.
It’s a simple app that responds to a few basic commands using APIs and predefined logic.

The goal of this project is to practice working with React, API calls, and basic chatbot behavior — not to be a full AI assistant

## Features
- Say hi
- Flip a coin (heads or tails)
- Roll a dice (1–6)
- Tell the current date (via an API)

## Built with 
- React

- JavaScript

- APIs (for date and random actions)

- Basic chatbot logic
